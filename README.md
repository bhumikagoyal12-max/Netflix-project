# 📊 Netflix Power BI Project

## 📌 Project Overview
This project is a Power BI dashboard built using Netflix dataset to analyze content trends, distribution, and insights.

---

## 📂 Files Included
- 📁 netflix project in power bi.pbix → Main dashboard file
- 📁 netflix_titles.csv.zip → Dataset used for analysis

---

## 🎯 Objectives
- Analyze Movies vs TV Shows distribution
- Identify top genres on Netflix
- Study content growth over years
- Analyze country-wise content production

---

## 📊 Key Insights
- Majority of content is Movies
- Significant growth after 2015
- USA contributes highest content
- Drama & Comedy are top genres
